# Meδia-Viewer

**Meδia-Viewer** is a dynamic multimedia viewer designed to make organizing and previewing media files a fast, unified, and seamless experience.

---

## 📌 Project Status & Notes

* **Language Note:** Source code comments are currently written in Spanish.
* **Build Guide:** Please note that `GUIA_COMPILACION.md` was created during the initial rapid prototyping phase and may not fully reflect the current state of the codebase. Use it as a general reference rather than a strict step-by-step guide.
* **Overview:** This is an active personal project created to streamline media organization—offering a faster, more efficient alternative to native file explorers when purging or sorting large sets of files.
